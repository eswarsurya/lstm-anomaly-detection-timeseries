# Public Output Evidence

This folder contains public-facing output evidence for the LSTM / time-series anomaly detection project.

## Files

- `dataset_profile.csv` summarises the public NYC taxi time-series sample.
- `top_anomaly_review_candidates.csv` gives the strongest deviation candidates from a simple rolling baseline.
- `daily_volume_sample.csv` shows daily volume aggregation for trend review.
- `model_output_summary.md` explains the LSTM anomaly detection direction and business value.

## Note

The deviation candidates are not presented as the final LSTM model output. They are included as public evidence to show the anomaly review style and the type of output the LSTM workflow is designed to support.
