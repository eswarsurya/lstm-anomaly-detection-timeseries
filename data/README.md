# Data Notes

This project uses an NYC taxi time-series sample with timestamp and observed value fields. The dataset is suitable for public review because it does not contain customer names, phone numbers, emails, or direct personal identifiers.

## Public Sample

`public_sample_nyc_taxi_timeseries.csv` contains the first 120 public-safe time-series records for quick GitHub review.

The local project also includes the fuller `nyc_taxi_sample.csv`, which is used for output summaries and model workflow development.
