# Report Summary

This project explores time-series anomaly detection using an LSTM-based workflow. The practical goal is to identify unusual changes in observed values over time, which is useful for monitoring traffic, demand, transactions, or sensor readings.

## Analysis Focus

The project focuses on preparing a time-series dataset, creating sequence windows, training or outlining a sequence model, reviewing anomaly candidates, and communicating unusual periods clearly.

## Public Evidence

The GitHub version includes public-safe NYC taxi time-series sample data, dataset profile outputs, anomaly candidate examples, and a clean public workflow script.

## Business Value

The project demonstrates machine learning thinking beyond static classification. It shows how sequence-based methods can support monitoring and early warning workflows.
